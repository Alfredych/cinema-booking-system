# Система онлайн-бронирования билетов в кинотеатр

## Описание проекта
Веб-приложение для бронирования билетов в кинотеатр с административной панелью управления. Проект реализован в рамках дипломной работы.

## Функциональные возможности

### Для гостей (неавторизованных пользователей):
- Просмотр расписания сеансов
- Просмотр информации о фильмах (постер, описание, длительность)
- Интерактивный выбор мест в кинозале
- Бронирование билетов
- Получение электронного билета с QR-кодом

### Для администраторов:
- Управление кинозалами (создание, редактирование, удаление)
- Управление фильмами (добавление, обновление, удаление)
- Настройка цен на обычные и VIP места
- Управление расписанием сеансов
- Закрытие/открытие залов для бронирования
- Для успешной авторизации используйте следующие данные: 
- - Логин: shfe-diplom@netology.ru
- - Пароль: shfe-diplom
